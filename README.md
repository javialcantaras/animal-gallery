## Getting started
### Requirements
![node (scoped)](https://img.shields.io/badge/node-v9.8.0-brightgreen.svg)

### Installation
```
$ git clone https://github.com/javialcantaras/animal-gallery.git
$ cd animal-gallery
$ npm install
```
### Run
```
$ npm start
```
### Test
```
$ npm test
```
